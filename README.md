djlockbook is a Django-based web application designed to simplify digital record management. It provides secure user registration and login functionality for staff and students, along with options to manage and store important records or reports efficiently.

This project demonstrates full-stack web development using Python (Django framework) with a responsive Bootstrap 5 frontend.
